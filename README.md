# AI Skill Gap Analyzer

## Description
AI Skill Gap Analyzer is a web app that helps users identify missing skills and improve their career readiness.

## Features
- User login/signup
- Skill analysis
- Personalized suggestions

## Tech Stack
- Python
- Flask
- HTML/CSS
- SQLite

## How to Run
1. Install dependencies:
   ```bash
   pip install -r requirements.txt

2.Run the application:
  python app.py

3. Open your browser at:
   http://localhost:5000
